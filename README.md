# 📚 StudyWell-AI

**Track. Analyze. Improve your study life with AI.**

## 🚀 Overview
StudyWell-AI is a machine learning-based application that helps students analyze their daily lifestyle and productivity. It predicts productivity levels and burnout risk based on user inputs and provides personalized suggestions to improve overall well-being.

---

## 🧠 Features
- 📊 Productivity Score Prediction  
- ⚠️ Burnout Risk Detection (Low / Medium / High)  
- 🧩 Behavioral Insights  
- 💡 Personalized Suggestions  

---

## 📥 Inputs
The system takes:
- Study hours  
- Sleep duration  
- Physical activity  
- Time spent with family  

---

## 📤 Outputs
- Productivity Score  
- Burnout Risk Level  
- Suggestions for improvement  

---

## 🛠️ Tech Stack
- Python  
- Machine Learning
